# ThinkPython2-CN
《ThinkPython2》中文译本, 希望能令你的Python学习之路简单一些。
当前, 笔者在重新修订简体中文版, 繁体版本目前差异较多, 请多担待.

## 一些话
希望你可以透过这本书,
了解一些编程理念. 
了解一些计算机世界以及网络世界.
学习一些软件工具.
掌握一些技巧,
得到一些助益.
